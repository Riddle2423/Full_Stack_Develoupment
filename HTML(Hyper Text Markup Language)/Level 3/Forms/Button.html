Buttons are of three types

  1. Submit

    used to sumbit form data.

  2. Reset

    Reset all form feilds to their default values.

  3. Normal Button

    Perform custom actons using JavaScript

we use attribute type to specify button type.

Submit button inside the form will take data where attribute action tells it to take the data.
