In HTML, entities are special codes used to display reserved characters or characters that are difficult to type directly in a webpage.
Why use HTML entities?
Some characters have special meanings in HTML. For example:

  • < starts a tag
  • > ends a tag
  • & begins an entity
If you want these characters to appear as text on the page, you must use entities.
